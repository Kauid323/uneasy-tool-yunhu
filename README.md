# uneasy-yunhu-tool
 ~ 不简单 ~ 难用的云湖工具

本系统为面向即时通讯场景的异构平台链接内容逆向解析与主动推送中间件，从中提取分享链接，通过动态逆向分析主流内容平台的Web端JavaScript加载逻辑，模拟其签名生成算法与请求构造流程，实现对分享链接原始结构化数据的非官方接口获取。系统以可视化不可交互组件形态存在，通过WebSocket与IM客户端建立长连接，采用Protocol Buffers完成协议层数据序列化与解析，最终将标准化内容推送至云湖聊天软件的目标会话。

if 你需要[ncm.js](bot-plugins/ncm.js)里的所有功能，请自行改依赖或者将所需的依赖添加到此文件夹的根目录。
